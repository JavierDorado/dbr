AYUDA DE USUARIO PARA EL DBR


 

A continuación se muestra una ayuda breve para manejar el DBR así como las teclas rápidas de acceso a las mismas.

 

Menú de archivo

 

Para acceder al menú de archivo, pulse F10 y a continuación cursor abajo. Una vez dentro, podrá moverse con las teclas de cursor arriba y cursor abajo. Si usa el ratón, haga click en el submenú Archivo para desplegarlo y ver las distintas opciones. Dentro de este menú hay cuatro opciones: Abrir libro, Buscar libro, Cerrar libro y salir.
-Abrir. Esta opción de menú abre un cuadro de diálogo en el que podrá seleccionar el libro DAISY que desee leer, indicando la ruta en la que se encuentra almacenado. También puede usar Control + A para acceder a este cuadro de diálogo.
-Buscar libro. Esta opción de menú muestra una lista con los últimos libros DAISY abiertos. Aquí puede seleccionar un libro DAISY para abrirlo. Al pulsar sobre el libro indicado, este se abrirá por la última posición de lectura. Si no se encontrase el libro indicado, se mostrará un mensaje de error indicándolo. También puede acceder a esta lista pulsando Control + B.
-Cerrar libro. Si pulsa esta opción de menú, el libro DAISy que esté en ese momento en pantalla se cerrará, quedándose la pantalla en blanco. Para realizar esta acción también puede pulsar la combinación de teclas Control + C.
-Salir. Si pulsa aquí, el DBR se cerrará. También puede pulsar Control + Q para salir de la aplicación.


Menú voz

 

Este menú le permite configurar el nivel de volumen de la voz del libro DAISY, así como activar o desactivar el audio. Para acceder a él, pulse F10 y desplázese con los cursores a la pestaña llamada Voz. Acto seguido pulse flecha abajo para entrar en el menú. Si usa el ratón, haga click sobre el submenú Voz para desplegarlo y ver su opción. La opción que este menú muestra es Ajustar volumen.
-Ajustar volumen. Esta opción le permite graduar el volumen del audio. Para ello, pulse sobre la opción de subir volumen o bajar volumen para subir o bajar el volumen respectivamente. También puede pulsar Control + V para subir el volumen o Control + Shift + V para bajarlo.
-Silenciar. Esta opción le permite activar o desactivar el audio del libro. Para ello, pulse sobre la opción o use la combinación de teclas Control + S.


Menú reproducción

 

 

El menú Reproducción le permite controlar la reproducción del libro DAISY y desplazarse a cualquier parte del libro. Para acceder a él, pulse F10 y a continuación desplácese con los cursores hasta la pestaña Reproducción. Pulse flecha abajo para abrir este menú, o haga click sobre el submenú Reproducción en la barra de menú para abrirlo. Este menú le mostrará las siguientes opciones: Reproducción-pausa, Detener, Navegación por capítulos, Navegación por páginas, Navegación por párrafos, Ir a...
-Reproducción-Pausa. Esta opción le permite activar o detener temporalmente la reproducción del libro DAISY dependiendo del estado actual de reproducción del libro DAISY. Pulse la tecla intro para cambiar el estado de reproducción. También puede usar la combinación de teclas Control + R.
-Detener. Esta opción detiene definitivamente la reproducción del libro DAISY y sitúa la lectura al principio del libro. Pulse la tecla intro sobre esta opción de menú para detener la lectura. También puede pulsar Control + D para realizar esta acción.
-Navegación por capítulos. Este menú cuenta con dos opciones para desplazarse al capítulo anterior o al siguiente. Para abrirlo, pulse cursor derecha. Seleccione la opción que desee y pulse intro. El DBR se situará al comienzo del capítulo indicado y comenzará a reproducirlo. Para ir al capítulo siguiente también puede pulsar la combinación de teclas Control + L y para ir al capítulo anterior la combinación de teclas Shift + control + L.
-Navegación por páginas. Esta opción de menú contiene un submenú con dos opciones: Página anterior y Página siguiente. Para abrirlo, pulse cursor derecha. Seleccione la opción que desee y pulse intro para acceder a la misma. EL DBR se situará al comienzo de la página indicada y empezará a reproducirla.
-Navegación por párrafos. Esta opción de menú le permite ir al párrafo anterior o siguiente. Para ello, abra el submenú con cursor derecha, sitúese en la opción deseada y pulse intro. El DBR comenzará la lectura desde el punto indicado. También puede pulsar Control + p para ir al bloque de texto siguiente o Shift + Control + P para ir al bloque de texto anterior.
-Ir a... Esta opción le permite ir a cualquier capítulo o página del libro. Para ello, pulse flecha derecha para abrir el submenú y pulse la opción deseada.
-Capítulo. Esta opción le permite ir a cualquier capítulo del libro. Al pulsar sobre ella, se desplegará una lista con los capítulos del libro. Seleccione el capítulo al que desea ir y pulse intro sobre el botón aceptar. El DBR se situará al principio del capítulo seleccionado y comenzará a reproducirlo. Si pulsa escape o el botón cancelar, en lugar de intro, cancelará la opción y volverá a la posición en la que se encontraba.
-Página. Esta opción le permite desplazarse a cualquier página del libro. Al pulsar intro sobre ella, aparecerá un cuadro en el que podrá introducir la página deseada. Al presionar el botón aceptar el DBR se situará al principio de la misma y comenzará a reproducirla. Si la página indicada no existe, el DBR mostrará un mensaje de error. Para cancelar esta opción, pulse el botón cancelar o escape en lugar de intro y volverá a la posición del libro en la que se encontraba.


Menú de marcas

 

Este menú le permitirá controlar la gestión de las marcas que usted realice sobre un libro en formato DAISY. Para acceder a él, pulse F10, desplácese con los cursores hasta el menú de marcas y pulse cursor abajo para desplegarlo. También puede hacer click sobre el submenú marcas con el ratón para desplegar las distintas opciones. El menú marcas tiene tres opciones: Establecer marca, Borrado de marcas y Listado de marcas.
-Establecer marca. Esta opción le permite crear una marca en el libro actualmente en reproducción. Para ello, pulse sobre la opción deseada, a la cuál también se puede acceder pulsando Control + E. A continuación se mostrará un cuadro de texto donde podrá introducir el nombre de la marca. Una vez introducido, pulse sobre el botón aceptar para guardar la marca, o pulse escape o el botón cancelar para cancelar el guardado de la marca.
Nota: tenga cuidado al introducir el nombre de la marca. Si la marca existe, el DBR la sobreescribirá.
-Borrado de marcas. Esta opción le permite borrar una marca, si existe al menos una, del libro actualmente en reproducción. Para ello, pulse sobre la opción deseada o la combinación de teclas Control + Z. A continuación se mostrará una lista con las marcas que dicho libro tiene. Elija la que quiera borrar y pulse sobre el botón aceptar. Si pulsa escape o cancelar, anulará el proceso.
- Listado de marcas. Esta opción le mostrará una lista con todas las marcas, si existe al menos una, del libro actualmente en reproducción y le permitirá desplazarse a la marca elegida. Para acceder al listado, pulse sobre la opción del menú o use la combinación de teclas Control + M. A continuación se mostrará un listado con todas las marcas. seleccione a la que quiera ir y pulse sobre el botón aceptar. Si pulsa escape o el botón cancelar, anulará el proceso.


Menú información

 

 

Este menú ofrece al usuario una serie de opciones de información sobre el libro y el tiempo de lectura del mismo, En concreto, cuenta con las siguientes opciones: Información general, Información de la traducción, Tiempo de lectura transcurrido y Tiempo total de lectura. Para acceder a este menú, pulse F10 y desplácese con los cursores hasta el submenú Información y pulse cursor abajo para abrirlo. Tambíen puede hacer click con el ratón sobre el submenú Infromación para desplegarlo.
-Información general del libro. Esta opción muestra información referente al autor, la editorial y aspectos similares. Para acceder a ella, pulse sobre esta opción. Se mostrará una ventana con la información solicitada. Una vez haya consultado la información, pulse intro o haga click con el ratón en el botón aceptar para volver al libro. Para acceder a esta información también puede pulsar Control + O.
-Información de la traducción. Esta opción de menú le ofrece información acerca de latraducción del libro a formato DAISY, tales como el traductor, la organización responsable o la fecha de la misma. Para acceder a ella, pulse sobre esta opción o la combinación de teclas Control + N. Una vez consultada la información, pulse intro o haga click con el ratón en el botón aceptar para volver al libro.
-Tiempo de reproducción transcurrido. Esta opción del menú le indica el tiempo de reproducción del libro DAISY contando desde el principio del mismo hasta la posición actual de lectura. Esta información se ofrece en el formato "hh:mm:ss" (horas, minutos, segundos). Para consultar esta opción del menú, pulse sobre la misma o presione la combinación de teclas Control + X. Una vez consultado el tiempo, pulse intro o haga click en el botón de aceptar para volver al libro.
-Tiempo total de lectura. Esta opción le muestra el tiempo total de reproducción que ocupa la obra, empezando desde el principio del libro hasta el final. El formato empleado es "hh:mm:ss" (horas, minutos, segundos). Para acceder a esta opción, pulse sobre la misma o presione la combinación de teclas Control + W. Una vez consultada, pulse intro o haga click sobre el botón aceptar para volver al libro.


Menú ayuda


Este menú le ofrece tanto la ayuda de usuario que aquí se muestra como otros datos de producción y de la licencia de uso. Para entrar a este submenú, pulse F10 y desplácese con los cursores hasta el mismo. También puede hacer click con el ratón en el submenú ayuda.
-Ayuda de usuario. Esta opción de menú le brinda la ayuda de usuario del DBR que aquí se muestra. Para acceder a la misma, pulse sobre esta opción. Al hacerlo, se mostrará en una ventana de Firefox esta ayuda. También puede acceder a la misma pulsando Control + U. Una vez haya consultado la ayuda de usuario, pulse Alt + F4 o cierre la ventana de Firefox para volver al DBR.
-Datos de producción. Esta opcción del usuario le ofrece información acerca del autor del DBR y del desarrollo del mismo. Para acceder a ella, pulse sobre la opción o pulse la combinación de teclas Control + Y. Al hacerlo, se abrirá una ventana de Firefox que le mostrará esta información. Una vez haya consultado la información, pulse Alt + F4 o cierre la ventana de Firefox con el ratón para volver al DBR.
-Licencia de uso. Esta opción del menú le ofrece información acerca de la licencia de uso y distribución del DBR. Para acceder a esta información, pulse sobre esta opción o use la combinación de teclas Control + G. Una vez haya entrado, se mostrará una ventana de Firefox con la información de la licencia. Para salir, pulse Alt + F4 o cierre la ventana de firefox con el ratón.
